Product Buttons
 
 
These buttons should be placed near each product description on your shopping pages.  Your currently selected button style is:
 
    You'll like this polo shirt.  $12.95   
 
    Here's a nice warm sweater.  $7.00   

If desired, you can select a different style for these buttons on the General Settings  screen.
 
When a customer presses a Product Button, we'll add that product to the customer's shopping cart.  We'll then give the customer an opportunity to select quantities and product varieties, if any.  The customer can then select more items to purchase, or pay and finish.
 
 
To install these Product Buttons:  Simply copy the HTML segments shown below, and paste them into your own HTML files after substituting the correct continue-page links.
 
Note the red text,  continue-page  in the code segments below.  In place of  continue-page,  insert the address of the Main Shopping page at your Web site.  (This usually starts with  http://  or  www.)   When the customer presses the  "Add More Items"  button from a CCNow page, we will send the customer back to the  continue-page  address that you specify.
 
For small shops, you'll probably want us to send the customer back to the page from where he/she originally clicked on this link.  Larger shops might want to send the customer to a search page in order to browse for his/her next product.
 
Special note for clients who are selling only one product:  You may leave out the  continue-page  parameter in the code segments below.  This tells our software not to display the  "Add More Items"  button.
 
Note that if you need to use the pound-sign (#) in your continue-page link, you will need to replace it with the special escape sequence:  !pound! .  This is necessary because of limitations in the HTTP character transfer protocol.
