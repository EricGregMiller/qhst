HOW TO NARROW YOUR SEARCH
You can use FIELD or in case you have more than one keyword, ANY or ALL.
(If you use only one keyword in the search, the ANY or ALL has no effect)

Leaving FIELD 'Blank' and choose:
ANY will search for any of the keywords in all fields, ex: 'AAA BBB' will result in 'AAA' or 'BBB' from whatever field.
ALL will search for all of the keywords in all fields, ex: 'AAA BBB' will result in 'AAA' and 'BBB' from whatever field.

If you additionally select a FIELD, it will narrow your search even further.
Select FIELD 'Name' and choose:
ANY will search for any of the keywords in the selected field only, ex: 'AAA BBB' will result in 'AAA' or 'BBB' from NAME field only
ALL will search for all of the keywords in the selected field only, ex: 'AAA BBB' will result in 'AAA' and 'BBB' from NAME field only.
